# GET /api/fast-api-trial-status

Check if the authenticated user has used their free Fast Spec trial.

## Overview

Solo users get one free Fast Spec generation before requiring a subscription. Use this endpoint to check trial eligibility.

## Endpoint

```
GET https://api.pre.dev/api/fast-api-trial-status
```

## Headers

```
x-api-key: YOUR_API_KEY
```

## Example Request

```bash
curl https://api.pre.dev/api/fast-api-trial-status \
  -H "x-api-key: YOUR_API_KEY"
```

## Response

### Trial Available

```json
{
  "hasUsedTrial": false,
  "usedAt": null
}
```

### Trial Used

```json
{
  "hasUsedTrial": true,
  "usedAt": "2025-10-01T14:30:00Z"
}
```

## Response Fields

| Field | Type | Description |
|-------|------|-------------|
| `hasUsedTrial` | boolean | `true` if trial has been used, `false` if available |
| `usedAt` | string | ISO timestamp when trial was used (null if not used) |

## Free Trial Details

### What's Included
- **One free Fast Spec** generation (10 generations)
- **No subscription required** for the first spec
- **Full functionality** - Same quality as paid specs
- **Completion email** with results and next steps

### Trial Limitations
- **One per user** - Cannot be reset or transferred
- **Fast Spec only** - Deep Spec requires subscription
- **No expiration** - Available until used

### After Using Trial

Once you've used your free trial:

1. **Subscribe to Solo Premium** for unlimited access
2. **Continue using the API** with your subscription
3. **Access all features** including Deep Spec and MCP integration

## Code Examples

### Check Trial Status

```bash
#!/bin/bash

API_KEY="YOUR_API_KEY"

# Check trial status
STATUS=$(curl -s https://api.pre.dev/api/fast-api-trial-status \
  -H "x-api-key: $API_KEY")

HAS_USED=$(echo $STATUS | jq -r '.hasUsedTrial')

if [ "$HAS_USED" = "false" ]; then
  echo "✅ Free trial available!"
  echo "You can generate one Fast Spec for free."
else
  echo "ℹ️  Free trial already used"
  USED_AT=$(echo $STATUS | jq -r '.usedAt')
  echo "Trial was used on: $USED_AT"
  echo "Subscribe to Solo Premium to continue: https://pre.dev"
fi
```

### Python Example

```python
import requests
from datetime import datetime

def check_trial_status(api_key: str) -> dict:
    """Check if free trial is available."""
    response = requests.get(
        'https://api.pre.dev/api/fast-api-trial-status',
        headers={'x-api-key': api_key}
    )
    response.raise_for_status()
    return response.json()

# Usage
api_key = "YOUR_API_KEY"
status = check_trial_status(api_key)

if not status['hasUsedTrial']:
    print("✅ Free trial available!")
    print("Generate your first Fast Spec for free.")
else:
    print("ℹ️  Free trial already used")
    if status['usedAt']:
        used_date = datetime.fromisoformat(status['usedAt'].replace('Z', '+00:00'))
        print(f"Trial was used on: {used_date.strftime('%Y-%m-%d %H:%M:%S')}")
    print("Subscribe to Solo Premium to continue.")
```

### JavaScript Example

```javascript
async function checkTrialStatus(apiKey) {
  const response = await fetch('https://api.pre.dev/api/fast-api-trial-status', {
    headers: { 'x-api-key': apiKey }
  });

  if (!response.ok) {
    throw new Error('Failed to check trial status');
  }

  return response.json();
}

// Usage
(async () => {
  try {
    const status = await checkTrialStatus('YOUR_API_KEY');

    if (!status.hasUsedTrial) {
      console.log('✅ Free trial available!');
      console.log('Generate your first Fast Spec for free.');
    } else {
      console.log('ℹ️ Free trial already used');
      if (status.usedAt) {
        const usedDate = new Date(status.usedAt);
        console.log(`Trial was used on: ${usedDate.toLocaleString()}`);
      }
      console.log('Subscribe to Solo Premium to continue.');
    }
  } catch (error) {
    console.error('Error checking trial status:', error.message);
  }
})();
```

## Integration with Application Flow

### Before Making First Request

```python
import requests

def smart_generate_spec(api_key: str, input_text: str) -> str:
    """Generate spec, checking trial status first."""

    # Check trial status
    trial_response = requests.get(
        'https://api.pre.dev/api/fast-api-trial-status',
        headers={'x-api-key': api_key}
    )

    if trial_response.status_code == 402:  # Insufficient generations
        print("No trial available and no subscription")
        print("Subscribe to Solo Premium: https://pre.dev")
        return None

    # Generate spec (trial will be used if available)
    spec_response = requests.post(
        'https://api.pre.dev/api/fast-spec',
        headers={'x-api-key': api_key, 'Content-Type': 'application/json'},
        json={'input': input_text, 'outputFormat': 'url'}
    )

    if spec_response.ok:
        result = spec_response.json()
        print(f"Spec generated: {result['output']}")
        return result['output']
    else:
        print(f"Generation failed: {spec_response.text}")
        return None

# Usage
spec_url = smart_generate_spec("YOUR_API_KEY", "Build a task management app")
```

## Best Practices

### User Experience
- **Check before prompting** - Let users know if trial is available
- **Clear messaging** - Explain trial limitations upfront
- **Smooth upgrade path** - Easy transition to subscription

### Error Handling
- **Handle 402 responses** - Insufficient generations (trial used)
- **Graceful degradation** - Show subscription options
- **Retry logic** - For temporary API issues
