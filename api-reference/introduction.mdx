## Quick Example

```bash
curl -X POST /api/fast-spec \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "input": "Build a task management app with team collaboration",
    "outputFormat": "url"
  }'
```
