## Authentication

The Architect API uses API key authentication with two methods:

### Solo Authentication

Use the `x-api-key` header with your user ID from [pre.dev](https://pre.dev) settings.

```bash
x-api-key: YOUR_USER_ID
```

### Enterprise Authentication

Use the `x-enterprise-api-key` header with your organization's API key.

```bash
x-enterprise-api-key: YOUR_ORG_API_KEY
```

  Get your API key from the [pre.dev dashboard](https://pre.dev) under Settings â†’ API Keys
</Info>

## Quick Example

```bash
curl -X POST /api/fast-spec \
  -H "x-api-key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "input": "Build a task management app with team collaboration",
    "outputFormat": "url"
  }'
```
