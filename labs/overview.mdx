## Why "Labs"?

We'd rather discover what's painful by feeling it ourselves than guess
at roadmaps. Every Labs product is something we needed, built, and use
in production. If it's good enough for our own agents, it's good
enough to ship.

## What to expect from each Labs product

- **An API** you can `curl` in 5 seconds
- **An MCP tool** for Claude Code / Cursor / any MCP-speaking agent
- **SDKs** in TypeScript and Python
- **A public benchmark** showing how we compare to the alternatives
- **Honest pricing** — no per-minute meter running while you sleep

## Roadmap

Browser Agents is the first launch. More Labs products will land here
as we productize them. If there's a tool you wish pre.dev had, tell
us — chances are we're already building it.
