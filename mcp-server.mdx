## What is pre.dev MCP?

The pre.dev MCP (Model Context Protocol) Server integrates seamlessly with Cursor and other MCP-compatible tools, allowing you to:

- **Generate project specifications** directly in your IDE
- **Create technical documentation** from natural language descriptions
- **Build architectural plans** for complex systems
- **Access pre.dev's AI** without switching contexts

## Quick Start

<Steps>
  <Step title="Get Your API Key">
    <Card title="Get API Key" icon="key" href="https://pre.dev/projects/playground">
      Visit the playground to get your API key
    </Card>
    
  </Step>
  
  <Step title="Install the MCP Server">
    <Tabs>
      <Tab title="One-Click Install">
        1. Visit [pre.dev/projects/playground](https://pre.dev/projects/playground)
        2. Click the **"Add to Cursor"** button
        3. Follow the prompts to complete installation
      </Tab>
      
      <Tab title="Manual Install">
        1. Open Cursor → Settings → Cursor Settings → **Tools & MCP**
        2. Click **"New MCP Server"**
        3. Add this configuration:

```json
{
  "mcpServers": {
    "predev": {
      "url": "https://api.pre.dev/mcp",
      "headers": {
        "x-api-key": "YOUR_API_KEY"
      }
    }
  }
}
```

        4. Replace `YOUR_API_KEY` with your actual API key
        5. Save and restart Cursor
      </Tab>
    </Tabs>
  </Step>
  
  <Step title="Test the Integration">
    Open Cursor and try this command:
    
    ```
    Use pre.dev to generate a spec for a task management app
    ```
    
    <Check>If successful, you'll receive a comprehensive project specification!</Check>
  </Step>
</Steps>

## Advanced Features

<AccordionGroup>
  <Accordion title="Context Integration" icon="link">
    Build on existing work by referencing previous specs or projects:
    
    ```
    Use pre.dev deep_spec to add payment processing 
    to this existing project: [existing_context_id]
    ```
    
    The MCP will automatically:
    - Fetch your existing context
    - Validate permissions
    - Generate new specs that build on your previous work
  </Accordion>
  
  <Accordion title="Documentation References" icon="book">
    Include external documentation for more accurate specs:
    
    ```
    Use pre.dev deep_spec to generate a spec for a React app, 
    referencing: https://react.dev/docs, https://tailwindcss.com/docs
    ```
  </Accordion>
  
</AccordionGroup>

## Troubleshooting

<AccordionGroup>
  <Accordion title="Authentication Failed" icon="triangle-exclamation">
    - Verify your API key is correct
    - Ensure you're using the right key (solo vs enterprise)
    - Check your subscription is active
  </Accordion>
  
  <Accordion title="Tool Not Found" icon="magnifying-glass">
    - Restart Cursor after adding MCP config
    - Verify the URL: `https://api.pre.dev/mcp`
    - Check JSON syntax in configuration
  </Accordion>
